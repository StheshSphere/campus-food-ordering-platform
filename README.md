# Campus Food Ordering Platform

A web-based campus food ordering platform built for the Software Design 2026 project.

## Sprint 1 Scope
Sprint 1 focuses on:
- User Verification
- Menu Management

## Roles
- Student
- Vendor
- Admin

## Core Features Implemented
- Google OAuth login via Supabase Auth
- Vendor onboarding with approval workflow
- Admin vendor approval and suspension
- Vendor menu management
- Menu image upload using Supabase Storage

## Tech Stack
- HTML, CSS, JavaScript
- Supabase Auth
- Supabase Database
- Supabase Storage
- GitHub Actions
- Azure Static Web Apps
